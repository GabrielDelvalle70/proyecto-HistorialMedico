# proyecto-HistorialMedico
Aplicación de escritorio desarrollada con Python para la gestión de historiales médicos de pacientes. Permite registrar, editar, buscar y eliminar información personal y médica de manera eficiente. El proyecto incluye una interfaz gráfica de usuario (GUI) diseñada con herramientas de Python y funciones que garantizan una gestión organizada y accesible de los datos.
